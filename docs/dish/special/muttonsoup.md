# 羊肉汤

**价格：13元**

>味道：2.5&#9733; / 5&#9734;
>
>性价比 4&#9733; / 5&#9734;

**唯一的作用是证明食堂有盐这种调味品。**

外观平平无奇，汤的表面浮一层辣椒油，下面是普通的羊汤，羊肉量适中，配两个馍作为主食。

**但是喝一口会发现汤里NaCl含量极高。** 仅仅喝两口或许还好，但一大碗浓NaCl溶液可不是所有人都可以接受的。表面的辣椒油更是折磨人的一把好手，又烫又咸又呛的汤让人不由怀疑自己到底是在cyyz食堂还是在西南方一个神秘的香料天堂之国。

搭配的馍泡在汤里会浸满了盐和辣椒油，加上本身口感就非同一般地弹滑（？），没有了碳水的口感，给人的正反馈也不佳，更建议手拿着一点一点的沾着汤吃。

羊肉没啥问题，能吃，好吃。建议只吃肉，汤抿两口就好。

_能喝完整碗汤的也是神人了。_
